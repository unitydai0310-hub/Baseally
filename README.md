# Baseally Website

Baseally corporate homepage prototype inspired by the structure and editorial rhythm of Hakuhodo's corporate site.

## Local preview

Open `index.html` directly in a browser for the fastest preview.

If your local environment allows a small static server:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Deploy

This is a static site. It can be deployed to Vercel from the project root after review.
